# Deepfake Detection System

This project is a web-based Deepfake Detection System that classifies images as REAL or FAKE using a deep learning model.

## Tech Stack
- Python
- TensorFlow (Xception)
- Flask
- HTML, CSS, JavaScript

## How to Run
1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   `python -m backend.app`
4. Open browser:
   http://127.0.0.1:5000

## Features
- Image upload
- Deepfake detection
- Confidence score output
