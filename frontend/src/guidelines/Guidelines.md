# Media Guidelines

These are practical usage notes for the current application behavior.

## Recommended input

- Use clear face images in `jpg`, `jpeg`, or `png`
- Prefer a single subject with the face visible
- Use reasonably sharp, well-lit images

## Current limitations

- Video files are not part of a completed inference pipeline yet
- Very small, blurry, or heavily compressed images may fail or produce inconclusive results

## Best results

- Upload one image at a time
- Wait for the current analysis to finish before starting another upload
